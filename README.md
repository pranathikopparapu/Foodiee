# 🍔 Foodiee – Online Food Ordering System

## 🚀 Live Demo

🌐 Frontend: https://foodapp-eq89bq6lt-pranathikopparapus-projects.vercel.app/
⚙️ Backend: https://foodiee-mfjc.onrender.com

---

## 📌 Project Overview

Foodiee is a full-stack **online food ordering web application** where users can browse food items, add them to cart, and place orders.
It also includes an **admin panel** for managing food items and orders.

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Axios
* React Router
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

### Deployment

* Frontend: Vercel
* Backend: Render

---

## ✨ Features

### 👤 User

* Browse food by category
* View food details
* Add to cart
* Place orders
* Login / Signup

### 🛠️ Admin

* Add / Edit / Delete food items
* Manage orders
* Dashboard view

## ⚙️ Installation (Local Setup)

### 1️⃣ Clone Repo

```bash
git clone https://github.com/your-username/Foodiee.git
cd Foodiee
```

---

### 2️⃣ Backend Setup

```bash
cd backend
npm install
npm start
```

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

## 🔑 Environment Variables

### Backend (.env)

```
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
```

---

### Frontend (.env)

```
REACT_APP_API_URL=https://foodiee-mfjc.onrender.com
```
